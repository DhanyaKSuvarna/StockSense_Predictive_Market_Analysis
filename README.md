# StockSense_Predictive_Market_Analysis
StockSense is an AI-driven decision-support tool for the analysis of stock markets through a combination of deep learning-based price prediction, technical trend analysis, and real-time visualization to help users make enlightened trading decisions.

# Features
- Fetching historical stock data through Yahoo Finance
- Visualization of the price trend using charts
- Technical indicators such as 100-day and 200-day moving averages
- Deep learning based price prediction model
- Evaluation of model performance using RMSE
- Buying / Selling / Holding Recommendation logics
- User-friendly web interface via Streamlit

# Model Information
The model applied for this project is trained separately for historical stock prices and stored as a Keras model file (`keras_model_universal.h5`). The Streamlit application uses this pre-trained model when it runs to produce predictions and trading signals.
